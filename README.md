# Hackathon Prompt

**Build something that helps people.**

## Prompt Requirements

### Core Requirements

🚀 Ship It: Live web application - Functioning live web application, reachable by anyone with an internet connection

### Optional Requirements (Pick at least 3)

- 📈 Insight Engine: Data-focused application - Charts, maps, dashboards that surface meaningful insights
- 🤖 For You: Recommendation system - Suggest content/actions based on user behavior or data
- 🔌 Plugged In: External data source - Integrates with APIs or third-party data
- 👥 Who Are You?: Multiple personas - Different user roles with distinct experiences/permissions (admin, user, etc.)
- 📍 Where Are You?: Location-based feature - Uses user location in a meaningful, non-trivial way
- 🌐 Mob Mentality: Crowd-sourced data - Users contribute and consume shared data
- 🔔 Ring Ring: Notifications / Alerts - Actions trigger alerts (email, SMS, in-app, etc.)
- 🧠 Make No Mistakes: AI / LLM incorporation - Uses AI for generation, classification, chat, etc.
- 💾 Memory Lane: Persistent state - Saves user actions, preferences, or history across sessions
- ⚡ Live Updates: Real-time updates - Application reflects changes dynamically without manual refresh
- 🔁 Connected Flow: Multi-step workflow - Users complete a sequence of actions that depend on each other
- 📜 Activity Feed: Tracking / logging - Tracks and displays user or system actions over time

## Requirements

**1. GitHub** <br>
Each team must submit a public GitHub repository. It must contain the following:

- `README.MD`
  - Writeup explaing the submission
- Code
  - Include all code written
  - Applicable documentation

**2. Website URL** <br>
Provide the URL for your live web application.

**3. YouTube** <br>
Each team must upload a video to YouTube.

- Serves as the offline "pitch" for your submission
- Video must be between 2-3 minutes in length
- Published on YouTube by submission deadline (unlisted or public)

**4. Presentation**
Teams selected as finalists (see schedule) will be required to do a live presenation to the judges' panel. _All teams must submit a slideshow_. This exact slideshow will be your presentation aid should your team be selected as a finalist.

- 3 minutes in length, 2 minutes allowed for Q&A
- Pitch and demo your project however you see fit
